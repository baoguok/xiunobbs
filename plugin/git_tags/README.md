### 插件简介

插件名：帖子标签

插件ID：git_tags

邀请广大的xiuno bbs开发者朋友，请前来协助完善这款标签插件。

这款标签插件，可以给标签任意加标签，传统方式纯用途的帖子标签插件。可以结合官方axiuno的标签插件共同使用，没有冲突，可以正常使用。
